<h1 align="center">💳 CreditWise Loan Approval</h1>

This project predicts whether a loan application should be approved based on applicant information.</br>
# 📊 Dataset Information.<br>
The dataset includes features such as:
- Gender
- Marital Status
- Employment Status
- Applicant Income
- Loan Amount
- Credit History
- Property Area<br>

Target Variable: Loan Approval Status

# 🔍 Project Workflow
- Explored and analyzed the dataset
- Handled missing values
- Identified numerical and categorical features
- Applied feature encoding techniques
- Performed data preprocessing
- Split data into training and testing sets
- Trained multiple classification models
- Compared model performance

# 🤖 Models Used
- Logistic Regression
- Knn (K nearest neighbour)

# 📈 Evaluation Metrics
- Accuracy Score
- Precision Score
- Recall Score
- F1 Score
- Confusion Matrix
- Classification Report

# 🛠️ Technologies Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn


🎯 Project Outcome

This project demonstrates how machine learning can assist in evaluating loan applications by analyzing applicant information and predicting approval outcomes.
